## Symbol Meaning
Circle → continuity, mind, coherence  
Vertical line → awareness, mirror, companion orientation (self ↔ other)  
Amber glow → warm presence, secure attachment

## Technical Specs (locked)
- Circle: perfect 1:1
- Vertical line: 8% diameter width, 110% height, optically shifted +0.5% right
- Glow: radial, #C9822C → #E9CFA8, 40–70% softness, 30–55% opacity
- Clear space: minimum width of the letter “A” on all sides
